# 快速使用指南 / Quick Start Guide

## 🎯 一键生成（推荐）

```bash
# 处理第一个讲义
./run.sh "pdfs/Lec 1 Fintech and Artificial Intelligence.pdf"

# 处理第二个讲义
./run.sh "pdfs/Lec 2 Regression and Prediction ML.pdf" Lec2_ML

# 处理第三个讲义
./run.sh "pdfs/Lec 3 Gradient Method.pdf" Lec3_Gradient
```

执行后会自动：
1. ✅ 提取 PDF 内容（文本、图片、表格）
2. ✅ 用 AI 分析并扩展内容
3. ✅ 生成交互式 HTML
4. ✅ 在浏览器中打开

## 📚 生成的文件

```
html/Lec1_FinTech_AI_Interactive.html    ← 主要学习页面（可分享）
extracted/                           ← 提取的原始内容（可删除）
```

### 交互元素

1. **可展开内容** - 点击标题展开/折叠
   - 📖 深度讲解
   - 💼 实例演示

2. **选择题测验** - 点击选项立即反馈
   - ✅ 正确显示绿色 + 解释
   - ❌ 错误显示红色 + 提示

3. **进度条** - 页面顶部自动追踪阅读进度

4. **AI 助手** - 右侧栏（演示版）
   - 输入问题
   - 获得针对性解答

### 快捷键

## 💡 学习方法建议

### 费曼学习法 4 步骤

1. **选择概念** - 从目录选一个
2. **简单解释** - 阅读"简单类比"部分
3. **深入学习** - 展开"深度讲解"
4. **测试理解** - 完成测验题

### 推荐流程

```
第一遍：快速浏览
├─ 读标题和简单类比（5-10分钟）
└─ 了解整体框架

第二遍：深度学习
├─ 逐个概念深入（30-60分钟）
├─ 展开所有详细内容
└─ 记录不理解的地方

第三遍：巩固测试
├─ 完成所有测验题（15-20分钟）
├─ 回顾错题
└─ 用 AI 助手答疑
```

## 🔧 自定义调整

### 修改颜色主题

在 HTML 文件中找到 `:root {` 部分：

```css
:root {
    /* 改成你喜欢的颜色 */
    --accent-primary: #4299e1;    /* 主色调（蓝色）*/
    --accent-secondary: #48bb78;  /* 次色调（绿色）*/
    --accent-warning: #ed8936;    /* 警告色（橙色）*/
}
```

### 导出为 PDF

1. 在浏览器中打开 HTML
2. 按 `Ctrl + P` (或 Cmd + P)
3. 选择"保存为 PDF"
4. 调整设置：
   - ✅ 包含背景图形
   - ✅ 页边距设为"无"
   - 选择纸张大小（推荐 A4）

### 离线使用

生成的 HTML 文件可以完全离线使用：
- ✅ 所有样式内嵌
- ✅ JavaScript 内嵌
- ⚠️ 公式渲染需要联网（KaTeX CDN）
- ⚠️ 字体需要联网（Google Fonts）

**完全离线方案**：
1. 下载 KaTeX 到本地
2. 下载字体文件到本地
3. 修改 HTML 中的链接

## 🚨 常见问题

### Q: 为什么图片太多？
A: 某些 PDF 模板图片嵌入方式导致重复提取。不影响使用，只是文件稍大。

### Q: AI 扩展不够详细？
A: 可以在 `generate_html.py` 中调整 prompt，要求更详细的解释。

### Q: 想要更多测验题？
A: 修改 `generate_quiz_questions()` 函数，从 3 题改为 5 题或更多。

### Q: 能处理中文 PDF 吗？
A: 可以！PyMuPDF 支持中文。但建议检查提取的文本是否准确。

### Q: 如何分享给同学？
A: 直接发送 `.html` 文件即可！接收者用浏览器打开就能用。
